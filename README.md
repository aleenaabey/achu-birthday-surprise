# Birthday Web Gift — Achu

Simple mobile-friendly HTML gift. Drop your photos and videos into the `assets/` folder and open `index.html` on the phone.

How to use
- Put media files in `assets/` (example names used: `photo1.jpg`, `photo2.jpg`, `video1.mp4`, `video2.mp4`).
- Open `index.html` in a mobile browser. If opening directly fails on iPhone, host it:

Local quick host (works with phone on same Wi‑Fi):

```bash
# from the project folder
python -m http.server 8000
# then open http://YOUR_PC_IP:8000 on the phone
```

Sharing options
- Zip the folder and send to the phone; open `index.html`.
- Push to GitHub and enable GitHub Pages for simple hosting.
